FBR POS CONNECTOR

This module will help you to connect Odoo POS with FBR. It will automatically/manaully post odoo pos orders to FBR and also handle refund/return pos orders.

Features

  • Automatically post POS orders to FBR while making order in pos session.

  • Manually post POS orders to FBR.

  • Refund/Return POS orders.

  • QR Code.

  • Cron job to post orders to FBR.

Demo Request

Cybat FBR POS Connector

Short demo shown in images below

  • Configure FBR URL, header authorization and POS ID.

  • Here you can enable/disable to post order to FBR automatically while placing order in pos session.
  • If you disable this check, pos orders will not be posted automatically to FBR from POS session.

  • You can post order manually to FBR.

  • FBR invoice number and QR code on pos receipt.

  • Enable this option to add POS FBR FEE automatically in pos receipt.

  • You can make a new product with checkbox(Is FBR FEE) True

  • You cannot add POS FBR FEE product more than one.

  • On click payment button POS FBR FEE will automatically be added into order.

  • You can filter orders which are not yet posted to FBR or failed for some reason/s.

  • Check FBR posting status(Post successfully or not), FBR Invoice Number and QR Code.

  • You can also refund/return order to FBR.

  • One refund/return order successfully you can see the status(Is returned) and FBR Return Invoice Number.

  • Cron job to post failed/not posted yet orders to FBR.

  • You can see the FBR invoice number in list view.
